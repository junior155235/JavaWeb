# JavaWeb
JavaWeb學習成果(仿104網站)

在現代社會，人力資源是任何組織成功的關鍵要素之一。無論是大型企業、小型公司還是創業公司，擁有一套強大的人力資源管理系統都至關重要,
因此此專題旨在藉由透過實作展現前後端的技術並為求職做準備。

開發目標:
    (1) 建立使用者會員系統
    (2) 記錄雇主刊登工作之情況以及求職者應徵之情況
    (3) 提供工作地區、工作性質等TAG以利使用者搜尋
